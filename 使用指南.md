# 竞赛信息管理系统 - 使用指南

## 系统启动

### 方法一：双击启动（推荐）
1. 进入 `G:\竞赛管理系统` 目录
2. 双击 `启动系统.bat` 文件
3. 等待系统启动完成
4. 浏览器访问 http://localhost:5000

### 方法二：命令行启动
```bash
cd G:\竞赛管理系统
.\venv\Scripts\activate
python backend\app.py
```

## 功能模块使用说明

### 1. 仪表盘

**功能**: 查看系统整体统计数据

- 竞赛总数：显示系统中所有竞赛数量
- 学生总数：显示系统中所有学生数量
- 采集数据：显示从网站采集的竞赛信息数量
- 参与记录：显示学生参与竞赛的记录数
- 最新采集数据：展示最近采集的10条信息

**操作流程**:
1. 点击左侧导航栏"仪表盘"
2. 查看各项统计数据
3. 浏览最新采集数据列表

### 2. 竞赛管理

**功能**: 管理竞赛信息（增删改查）

**添加竞赛**:
1. 点击左侧导航栏"竞赛管理"
2. 点击"添加竞赛"按钮
3. 填写竞赛信息：
   - 竞赛名称（必填）
   - 官网链接
   - 描述
   - 类别
   - 状态（进行中/即将开始/已结束）
   - 开始日期
   - 结束日期
4. 点击"确认"保存

**编辑竞赛**:
1. 在竞赛列表中找到目标竞赛
2. 点击"编辑"按钮
3. 修改竞赛信息
4. 点击"确认"保存

**删除竞赛**:
1. 在竞赛列表中找到目标竞赛
2. 点击"删除"按钮
3. 确认删除操作

### 3. 学生管理

**功能**: 管理参赛学生信息

**添加学生**:
1. 点击左侧导航栏"学生管理"
2. 点击"添加学生"按钮
3. 填写学生信息：
   - 姓名（必填）
   - 年级
   - 专业
   - 电话
   - 邮箱
4. 点击"确认"保存

**编辑学生**:
1. 在学生列表中找到目标学生
2. 点击"编辑"按钮
3. 修改学生信息
4. 点击"确认"保存

**删除学生**:
1. 在学生列表中找到目标学生
2. 点击"删除"按钮
3. 确认删除操作

### 4. 数据采集

**功能**: 从 cpipc.acge.org.cn 网站采集最新竞赛信息

**操作流程**:
1. 点击左侧导航栏"数据采集"
2. 点击"立即采集"按钮
3. 等待采集完成
4. 查看采集结果列表：
   - 标题
   - 摘要
   - 来源链接
   - 发布日期
   - 采集时间

**说明**:
- 系统会自动解析网页内容
- 生成中文摘要
- 避免重复数据
- 采集结果可在仪表盘中查看

### 5. 系统日志

**功能**: 查看系统运行日志

**操作流程**:
1. 点击左侧导航栏"系统日志"
2. 查看日志列表：
   - 日志级别（信息/警告/错误）
   - 日志消息
   - 模块名称
   - 时间戳

**日志级别说明**:
- 信息 (蓝色): 正常操作记录
- 警告 (黄色): 需要注意的情况
- 错误 (红色): 操作失败或异常

## 常见问题

### Q1: 系统无法访问？
A: 检查系统是否启动成功，查看命令行是否有错误信息。

### Q2: 数据采集失败？
A: 检查网络连接是否正常，目标网站是否可访问。

### Q3: 数据丢失？
A: 数据库文件位于 `G:\竞赛管理系统\data\competition.db`，请勿删除。

### Q4: 如何备份数据？
A: 复制 `data\competition.db` 文件到其他位置即可。
